Щоденник артеріального тиску, пульсу, ваги та ІМТ. Вимірювання зберігаються на вашому сервері — без хмарних посередників.

  • Ранкові та вечірні виміри по кожній руці
  • Класифікація тиску WHO/ESH 2023

  • PDF-довідка для лікаря
  • Push-нагадування про виміри
  • GDPR: дані лишаються тільки у вас
🔒 Політика конфіденційності

Хто обробляє дані
Цей застосунок є особистим інструментом відстеження здоров'я. Власник сервера є єдиним контролером ваших даних.

Які дані збираються
Email, ім'я, дата народження, зріст, показники артеріального тиску, пульс, вага, нотатки до записів.

Мета обробки
Виключно для відображення вашої особистої медичної статистики. Дані не передаються третім особам.

Правова підстава (GDPR ст. 9(2)(a))
Обробка спеціальної категорії персональних даних (здоров'я) здійснюється на підставі вашої явної згоди, наданої при реєстрації.

Зберігання
Дані зберігаються на приватному сервері. Шифрування транспорту — HTTPS. Дані зберігаються до видалення акаунту.

Ваші права (GDPR)

  • Доступ — ви бачите всі свої дані в застосунку
  • Портативність — експорт у JSON / CSV (Журнал → Експорт)
  • Видалення — кнопка "Видалити акаунт" у профілі видаляє всі записи безповоротно
  • Відкликання згоди — видалення акаунту відкликає згоду

Питання щодо обробки даних направляйте власнику сервера.
